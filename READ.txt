
* 프로그램 내용
  : 식당과 고객 사이의 주문 서비스


* 프로그램 절차
  1. 서버가 켜짐
  2. 클라이언트가 서버에 접속 -> 서버는 메뉴를 보냄
  3. 클라이언트는 받은 메뉴로 UI 구성 -> 메뉴 선택
  4. 주문 구조체를 서버에게 전송
  5. 서버는 받은 주문으로 UI 구성


* 메뉴 구조체(menu)
  - menu_len / int / 클라이언트에서 메뉴 주문서를 동적으로 할당하여 만들기 위함
  - menu_str / char[512] / 메뉴 출력을 위함

* 주문 구조체(order)
  - no / int / 주문 번호(서버에서 할당)
  - name / char[20] / 주문자 이름
  - phone / char[20] / 주문자 번호
  - add / char[50] / 주문 주소
  - order / int[30][2] / 주문서 내용 (메뉴 번호 | 수량)
  - time / time_t / 주문 시간
* 메뉴와 주문 구조체는 별도의 헤더 파일에 입력하여 이를 사용


* 서버와 클라이언트는 fork를 사용
  - 서버: 클라이언트가 접속할 때마다 child process 생성
  - 클라이언트: read와 write를 따로 시행
